File Upload & Management

Welcome to File Up, a Spring Boot application for uploading, managing, and downloading files. This application supports tabular data (Excel, CSV) and non-tabular files (e.g., PDFs), with features to view, download, and delete files.

This is a static landing page hosted on GitHub Pages. The application itself requires a server to run and cannot be fully hosted here.

Run the Application Locally

Follow these steps to run the Manpower application on your local machine:

  1. Clone the Repository: 
    git clone https://github.com/your-username/manpower.git
  2. Navigate to the Project Directory: 
    cd manpower
  3. Build the Project: 
    mvn clean install
  4. Run the Application: 
    mvn spring-boot:run

Once the application is running, use the "Access the App" button above to visit the app at http://localhost:8080/data/upload.

Requirements: Ensure you have Java 11+ and Maven installed on your system.
